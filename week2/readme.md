# Linked List
![](/image/螢幕截圖%202019-10-18%2014.15.51.png)
### array需要整塊連續的記憶體空間，但lined-list可以由一個一個空間拼湊起來。

## 為什麼要使用Linked List ?
### 1、在記憶體位置可為不連續的。
### 2、能夠頻繁的刪除及新增資酪。
### 3、空間分配適當，不會浪費空間。
### 4、Block Chain 的應用。
### 5、能夠查詢走訪所有node。
