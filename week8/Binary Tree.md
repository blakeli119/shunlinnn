# Binary Tree（二元樹）
# 基本介紹
### 一般的樹(Tree)對於樹上的node之child（子結點）數目沒有限制，因此可以有很多個。

### Binary Tree（二元樹)則限制每個node只能有兩個child，兩個子結點分別稱為，左子結點與右子節點，而同時連結這兩個數的，稱為他們的父節點。
