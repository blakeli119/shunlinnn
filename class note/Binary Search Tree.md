# Binary Search Tree(BST、二元搜尋數)
## 基本概念
### 1、若任意節點的左子樹不空，則左子樹上所有節點的值均小於它的根節點的值
### 2、若任意節點的右子樹不空，則右子樹上所有節點的值均大於它的根節點的值
### 3、任意節點的左、右子樹也分別為二元搜尋樹
### 4、沒有鍵值相等的節點

## 功能
### Search（查詢）：尋找某特定資料，是否存在資料結構中。
### insert（插入）：加入新的數值，與search的走訪過程相似。
### delet（刪除）：刪除資料庫中特定的數。
### modify（修改）：將原有的數移除換為一個新的數值。

## 時間複雜度
### space（空間）：平均（O(n)）、最差（O(n)）
### search（搜尋）：平均（O(log n)）、最差（O(n)）
### insert（插入）：平均（O(log n)）、最差（O(n)）
### delet（刪除）：平均（O(log n)）、最差（O(n)）
