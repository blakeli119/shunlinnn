# Insertion Sort（插入排序法）
## 執行步驟
### 1、第一個，也就是最左邊的數字直接做為已排序的頭。
### 2、將下一個值(右邊的數)作為基準值。
### 3、與所有已排序中的數字比對(由右至左)。
### 4、移動所有比基準值大的數字。
### 5、插入基準值。
### 6、回到第2個步驟，重複執行。
